### 阶乘预处理

`阶乘以及其逆元预处理`

```c++
ll fac[N + 3], invfac[N + 3];
ll C(ll n, ll m){
    ll ans = (fac[n] * invfac[m] % mod) * invfac[n - m] % mod;
    return ans;
}
fac[0] = 1;
for(int i = 1; i < N; i++)	fac[i] =(fac[i - 1] * i) % mod;
invfac[N - 1] = binpow(fac[N - 1], mod - 2);
for(int i = N - 2; i >= 0; i--)	invfac[i] = invfac[i + 1] * (i + 1) % mod;
```
